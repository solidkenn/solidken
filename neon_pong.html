📋 CHANGELOG

VERSION X ORIGIN
Made a functioning Pong game
VERSION 0.10
Fixes & Changes
Fixed the scoring bug — score was jumping to 7 instantly
Dynamic ball color — ball color changes when a paddle hits it
VERSION 0.11
Fixes & Changes
Renamed the game to Neon Pong
Changed Player 1 paddle to yellow for visibility; applies to dynamic ball color effect
Enhanced ball speed and collision during rallies
Fixed Enter key bug — after starting, Enter was accidentally restarting the game
Added Escape key as a Pause key
Added Deuce feature — 2-point lead required to win; advantage banner shows who leads
VERSION 0.12
New Features
Settings menu added in the start overlay, accessible any time from the Pause Menu
Paddle color customization — color wheel picker per player, live preview on canvas
Ball speed controls — three sliders: Hits Before Ramp, Ramp Rate Per Hit, Top Speed Cap
Sound effects via Web Audio API — Paddle Hit, Wall Bounce, Score, Deuce, Advantage, Win, Pause
Sound effects toggle button in the footer to mute/unmute at any time
Version label added to footer
VERSION 0.20 BETA
Visual & Gameplay Changes
4 playable stages: Neon, Grass, Clay, Classic — each with unique court design and ball rendering
Two game modes: Arcade (first to 7) and Competitive (2-point lead rule)
Main Menu with stage, mode, and settings selection before each match
Paddle color customization disabled on Grass, Clay, and Classic stages
Settings removed from Pause Menu — now only accessible from Main Menu
Quit to Menu button added to Pause overlay
Ball highlight added in Neon stage — white specular glow keeps dark-colored balls visible
VERSION 0.30 PUBLIC BETA
Changes
UI polished on desktop view — readability improvements across all text
1 Player Mode added with AI opponent
Difficulty selector: Easy, Medium, Hard — each with tuned AI behavior
Modified default ball speed ramp values
VERSION 0.35 MOBILE
Changes
Added mobile compatibility for tablets and phones
Touch drag zones for paddle control — left zone for P1, right zone for P2
Portrait lock screen prompts device rotation
On-screen Pause button for touch devices
Paddle touch strips added as a kids-friendly narrow control zone
AI rebalanced for Medium and Hard difficulty
VERSION 0.40 D-PAD SUPPORT
Changes
UI Polishing in both Desktop and Mobile View
Fix the Tail Ball issue in Classic Stage
Added controller and D-pad support for gameplay controls
