Changelog
HearMe Bingo — by solidkenn
Build 0.1.4 Current
Improved voice call cadence for clarity : call format is now “Letter B, 69, B, 69”.
Added a Repeat Last Call button to replay the latest spoken call.
Added a Voice Test button to preview speech before or during the game.
Added reset confirmation to prevent accidental game resets.
Improved Recap speech to explicitly say “Letter” for each B-I-N-G-O group.
Disabled Repeat Last Call and Recap while the caller is speaking.
Updated the welcome message to avoid first-word clipping on some voices.
Updated app version in header to v0.1.4.
Build 0.1.3
Added a Mute button on the volume slider for quick in-game silence.
Updated Auto Call speed presets to Slow (8s), Normal (5s), and Fast (2s) for a faster game pace.
Improved letter speech : now uses full phonetic names: Bee, Aye, En, Gee, Oh for clearer announcements.
Improved column distribution in the draw sequence : no more than 3 consecutive numbers from the same letter column.
Added build version v0.1.3 and “by solidkenn” signature to the header.
Added this Changelog popup.
Build 0.1.2
Improved the Called Numbers history : now shows the last 10 called numbers with a “View All” popup.
Replaced the speed interval slider with Slow, Normal, and Fast preset buttons.
Fixed a bug where Auto Call would not start when the Start button was clicked with Auto Call pre-enabled.
Improved the Recap voice announcement for better clarity.
Added a Stop button on Recap to cancel reading mid-recap : merged into a single toggle button.
Build 0.1.1
Added Male and Female voice selection for the Voice Caller.
Improved number announcement cadence : letter, number, then letter + spoken word.
Improved RNG using cryptographically strong randomness.
Added a volume slider to control Voice Caller loudness.
Fixed the Next button to be disabled while the Caller is still speaking.
Improved the Called Numbers history UI with better layout and readability.
Added a speed interval slider for the Auto Call feature.
Build 0.1.0
Created a fully functioning Bingo Caller.
Added a Voice Caller with text-to-speech announcement.
Added a Reset button to restart the game at any time.
Added a Called Numbers history panel.
Added a Recap button to read back all called numbers.
Added Auto Call : automated number drawing on a set interval.
Version X Prototype
HearMe Bingo prototype created — initial concept and layout.
